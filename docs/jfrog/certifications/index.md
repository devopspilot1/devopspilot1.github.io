## Associate JFrog Artifactory Certification

**Validates your ability to manage and deploy artifacts using JFrog Artifactory.**

### Exam Details

| Property | Value |
|---|---|
| **Questions** | 48 Multiple Choice |
| **Lab** | Hands-on practical environment |
| **Time** | 60 min (exam) + 60 min (lab) |
| **Passing Score** | 80% on exam + successful lab task completion |
| **Attempts** | Up to 2 attempts on theoretical exam |
| **Language** | English |
| **Price** | $200 |

### Topics Covered

- Artifactory functionality and architecture
- Configuring build tools (Maven, Gradle, npm, Docker)
- Managing development and release processes
- Utilizing and configuring repositories (Local, Remote, Virtual)
- Security, permissions, and access control
- Role-Based Base Permissions (RBv2)
- Troubleshooting resolution errors
- Unidirectional artifact transfer (Distribution)
- Setting up JFrog Xray
