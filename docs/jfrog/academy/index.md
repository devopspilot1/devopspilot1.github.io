## JFrog Academy Highlights

### Free Self-Paced Courses

JFrog Academy offers free e-learning modules that complement hands-on practice:

- **JFrog Artifactory Foundations** ‚Äî Repos, builds, security basics
- **JFrog Xray Essentials** ‚Äî Scanning, policies, SBOM
- **JFrog CLI Hands-on** ‚Äî Practical CLI exercises
- **JFrog Platform Overview** ‚Äî All products at a glance

### Instructor-Led Training (ILT)

For teams needing structured training:

- **Artifactory for Developers** ‚Äî 1-day course on repos, build tools, CLI
- **Integrating Artifactory in a Distributed DevOps Process** ‚Äî Pre-requisite recommended for HA/DR cert
- **JFrog Security Training** ‚Äî Deep-dive into Xray, Curation, JAS

### Virtual Labs

Many courses include **sandboxed lab environments** ‚Äî a full JFrog Platform instance pre-configured for exercises. You don't need your own JFrog account to complete labs.
