<quiz>
Which command combines two branches while creating a new commit?
- [x] git merge
- [ ] git combine
- [ ] git join

`git merge` joins two or more development histories together.
</quiz>

<quiz>
Which command temporarily saves your changes without committing them?
- [x] git stash
- [ ] git save
- [ ] git pause

`git stash` temporarily shelves (or stashes) changes you've made to your working copy so you can work on something else, and then come back and re-apply them later.
</quiz>

<quiz>
What is the difference between git fetch and git pull?
- [x] fetch downloads changes, pull downloads and merges
- [ ] fetch merges changes, pull only downloads
- [ ] there is no difference

`git fetch` only downloads data, while `git pull` downloads and then merges the data into your current branch.
</quiz>

<quiz>
Which command is used to discard changes in the working directory?
- [x] git restore
- [ ] git remove
- [ ] git delete

`git restore` restores working tree files. It can be used to discard uncommitted changes.
</quiz>

<quiz>
How do you view the commit history?
- [x] git log
- [ ] git history
- [ ] git commits

`git log` shows the commit logs.
</quiz>
