<quiz>
Which command initializes a new Git repository?
- [x] git init
- [ ] git start
- [ ] git create

`git init` creates a new Git repository. It can be used to convert an existing, unversioned project to a Git repository or initialize a new, empty repository.

Which command is used to stage files?
- [x] git add
- [ ] git stage
- [ ] git push

`git add` adds content from the working directory to the staging area (or "index") for the next commit.

Which command creates a new branch?
- [x] git branch [name]
- [ ] git new [name]
- [ ] git create-branch [name]

`git branch` is used to create, list, and delete branches.

Which command is used to download a repository from a remote source?
- [x] git clone
- [ ] git copy
- [ ] git download

`git clone` is used to create a copy of a specific repository or branch within a repository.

Which command shows the status of changes as untracked, modified, or staged?
- [x] git status
- [ ] git check
- [ ] git info

`git status` displays the state of the working directory and the staging area.
