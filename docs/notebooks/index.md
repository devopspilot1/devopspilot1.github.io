# Jupyter Notebooks

Welcome to the Jupyter Notebooks section of DevopsPilot. Here you'll find interactive tutorials and examples for various AWS services and tools.

## Available Notebooks

### AWS Bedrock AgentCore Samples

The following notebooks are from the [Amazon Bedrock AgentCore Samples](https://github.com/aws/amazon-bedrock-agentcore-samples) repository.

**License:** Apache License 2.0

These tutorials cover:
- **AgentCore Runtime**: Hosting agents with various frameworks
- **AgentCore Gateway**: Gateway configuration and deployment
- **AgentCore Identity**: Identity and authentication
- **AgentCore Memory**: Memory management
- **AgentCore Tools**: Tool integration and custom tools
- **AgentCore Observability**: Monitoring and observability
- **AgentCore Evaluations**: Evaluation frameworks
- **AgentCore Policy**: Policy configuration
- **AgentCore E2E**: End-to-end examples

## How to Use These Notebooks

1. Each notebook is interactive and can be run in Jupyter environments
2. Code examples are ready-to-use and tested
3. Follow along with the explanations and output
4. Modify and experiment with the code

## Installation Requirements

Most notebooks require:
- Python 3.10+
- boto3 (AWS SDK for Python)
- bedrock-agentcore libraries
- Additional dependencies as specified in each notebook
